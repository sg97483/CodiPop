# Google Cloud Console - SHA-1 추가 방법

## 📋 현재 상황

스크린샷을 보면:
- **SHA-1 인증서 디지털 지문** 필드에 업로드 키 SHA-1이 이미 입력되어 있음
- `2D:FB:8F:C4:67:A6:AF:89:48:46:3D:C9:B9:32:43:65:83:14:E4:32`

## ✅ SHA-1 추가 방법

### 방법 1: 같은 필드에 여러 개 입력 (가장 일반적)

1. **SHA-1 인증서 디지털 지문** 필드를 클릭
2. 기존 SHA-1 뒤에 **Enter 키**를 눌러 새 줄 추가
3. 또는 기존 SHA-1 뒤에 **쉼표(,) 또는 줄바꿈**으로 구분하여 추가
4. 앱 서명 키 SHA-1 입력:
   ```
   05:BD:0D:9D:E3:CA:C1:A8:8E:B2:96:0F:2F:04:EC:1D:38:B6:B1:06
   ```
5. **저장** 버튼 클릭

### 방법 2: 필드 옆의 "+" 또는 "추가" 버튼 찾기

1. **SHA-1 인증서 디지털 지문** 필드 옆을 확인
2. 작은 **"+" 아이콘** 또는 **"추가" 버튼** 찾기
3. 클릭하면 새 입력 필드가 나타남
4. 앱 서명 키 SHA-1 입력:
   ```
   05:BD:0D:9D:E3:CA:C1:A8:8E:B2:96:0F:2F:04:EC:1D:38:B6:B1:06
   ```
5. **저장** 버튼 클릭

### 방법 3: 기존 SHA-1 수정하여 두 개 모두 포함

1. **SHA-1 인증서 디지털 지문** 필드를 클릭
2. 기존 SHA-1을 유지한 채로
3. **Enter 키**를 눌러 새 줄 추가
4. 앱 서명 키 SHA-1 추가:
   ```
   2D:FB:8F:C4:67:A6:AF:89:48:46:3D:C9:B9:32:43:65:83:14:E4:32
   05:BD:0D:9D:E3:CA:C1:A8:8E:B2:96:0F:2F:04:EC:1D:38:B6:B1:06
   ```
5. **저장** 버튼 클릭

## 🎯 최종 결과

두 개의 SHA-1이 모두 등록되어야 합니다:

1. **업로드 키 SHA-1** (이미 있음):
   ```
   2D:FB:8F:C4:67:A6:AF:89:48:46:3D:C9:B9:32:43:65:83:14:E4:32
   ```

2. **앱 서명 키 SHA-1** (추가 필요):
   ```
   05:BD:0D:9D:E3:CA:C1:A8:8E:B2:96:0F:2F:04:EC:1D:38:B6:B1:06
   ```

## 📝 단계별 가이드

1. **SHA-1 필드 클릭**
   - 현재 입력된 SHA-1이 선택되거나 커서가 나타남

2. **새 SHA-1 추가**
   - Enter 키를 눌러 새 줄 추가
   - 또는 필드 옆의 "+" 버튼 클릭

3. **앱 서명 키 SHA-1 입력**
   ```
   05:BD:0D:9D:E3:CA:C1:A8:8E:B2:96:0F:2F:04:EC:1D:38:B6:B1:06
   ```

4. **저장 버튼 클릭**
   - 페이지 하단의 "저장" 버튼 클릭

## ⚠️ 참고사항

- 두 개의 SHA-1을 모두 등록해야 합니다
- 업로드 키 SHA-1: 개발/테스트용
- 앱 서명 키 SHA-1: 실제 배포 앱용 (중요!)
